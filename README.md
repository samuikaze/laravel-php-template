# Laravel 11 PHP 8.3 Template

這是一個使用 PHP 8.3 版的 Laravel 11 模板專案，用於快速起始新的專案用

## 使用說明

使用上完全遵照 Laravel 文件說明進行開發與部署即可

> 請注意，此版本使用 [FrankenPHP](https://frankenphp.dev/) 讓應用程式提供服務。
